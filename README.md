## Технологии
- Используется Django последней LTS-версии. На момент разработки это версия 3.2.20

## Использование
Установите Django последней LTS-версии:
```sh
$ pip install Django==3.2.20
```
Запустите виртуальное окружение:
```sh
$ source venv/bin/activate
```
И запустите сервер:
```sh
$ ./manage.py runserver
```
